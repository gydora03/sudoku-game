# sudoku

The sudoku game.

## Requirements

Building the project requires JDK 11 or later and [Apache Maven](https://maven.apache.org/).

## How to run
```
git clone https://github.com/gydora03/sudoku-game.git
cd sudoku-game/
mvn package
java -jar ./target/sudoku-game-1.0-jar-with-dependencies.jar
```
